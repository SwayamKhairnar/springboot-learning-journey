Spring Boot Learning Journey

This repository contains my step-by-step learning of Spring, Spring Boot, and JPA.
Each folder represents a stage in my understanding, starting from core concepts
and moving towards building a complete CRUD application.

Structure:
- 01-basics → Dependency Injection and IoC fundamentals
- 02-spring-vs-springboot → Understanding differences through small demos
- 03-layered-architecture → Controller, Service, Repository pattern
- 04-crud-with-jpa-h2 → CRUD application using Spring Boot, JPA, and H2
- notes → Personal learning notes and observations

Tech stack used:
- Java
- Spring Framework
- Spring Boot
- Spring Data JPA
- H2 Database

This repository is actively being updated as I continue learning.
